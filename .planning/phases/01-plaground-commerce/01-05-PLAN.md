**A. API routes (apps/api/src/routes/v1/uploads/index.ts)**

Register two routes on `/api/v1/uploads`:

`POST /api/v1/uploads/initiate` — authenticated (requireAuth), requires permission
`uploads.write`. Body: `{ filename: string, mimeType: string, fileSize: number }`. Zod validation:
- `filename`: non-empty string, max 260 chars
- `mimeType`: one of `'model/stl'`, `'model/x.stl-binary'`, `'model/x.stl-ascii'`,
  `'application/vnd.ms-3mfdocument'`, `'application/octet-stream'` — accept on mime OR rely on
  extension from filename
- `fileSize`: positive integer, max `104857600` (100MB)

Logic:
1. Determine format from filename extension: `.stl` → `STL`, `.3mf` → `THREE_MF`, `.obj` → `OBJ`,
   `.step`/`.stp` → `STEP`. Reject any other extension with 422 `{ code: 'UNSUPPORTED_FORMAT', message: 'We accept STL, 3MF, OBJ, and STEP files.' }`.
2. Create an `Upload` record via prisma with fields: `tenantId`, `userId`, `filename`, `mimeType`,
   `fileSize`, `format`, `status: 'PENDING'`, `storageKey: \`uploads/${tenantId}/${crypto.randomUUID()}-${filename}\``.
3. Generate a presigned S3 PUT URL with `@aws-sdk/s3-request-presigner`:
   `getSignedUrl(s3Client, new PutObjectCommand({ Bucket: S3_BUCKET_NAME, Key: upload.storageKey, ContentType: mimeType, ContentLength: fileSize }), { expiresIn: 300 })`.
4. Return `{ uploadId: string, uploadUrl: string, storageKey: string, expiresAt: ISO8601 }`.

`POST /api/v1/uploads/complete` — authenticated, requires `uploads.write`. Body:
`{ uploadId: string }`.
1. Find Upload by id where `userId === request.sessionData.userId` and `status === 'PENDING'`.
2. Set `status: 'SCANNING'`, update `updatedAt`.
3. Enqueue BullMQ job: `uploadScanQueue.add('scan', { uploadId: upload.id, storageKey: upload.storageKey, format: upload.format }, { attempts: 3, backoff: { type: 'exponential', delay: 2000 } })`.
4. Return `{ uploadId, status: 'SCANNING' }`.

new Worker('upload-scan', async (job) => {
  const { uploadId, storageKey, format } = job.data
  // 1. Download file from S3 using GetObjectCommand to a temp Buffer (max 100MB)
  // 2. Validate: confirm actual file magic bytes match declared format
  //    STL binary: first 80 bytes are header; binary if bytes 80-83 contain triangle count
  //    3MF: starts with PK\x03\x04 (ZIP magic bytes)
  //    OBJ: starts with #, v, vn, or vt ASCII markers

  // 4. If format valid: set Upload.status = 'ACCEPTED', then:
  //    - Create ModelFile record with: uploadId, format, fileSize, status: 'PENDING_ANALYSIS'

**A. Model analysis worker (apps/worker/src/processors/model-analyze.ts)**

BullMQ Worker consuming `'model-analyze'` queue, concurrency 2 (geometry parsing is CPU-intensive):

```typescript
new Worker('model-analyze', async (job) => {
  const { uploadId, modelFileId, storageKey } = job.data
  // 1. Download file from S3 to Buffer
  // 2. Parse geometry depending on format:
  //    STL binary: read triangle count from bytes 80-83 (LE uint32),
  //      iterate triangles (50 bytes each) to extract vertex positions,
  //      compute bounding box (min/max x,y,z) and approximate volume
  //      using signed tetrahedron volume method (sum of cross-product contributions)
  //    3MF: unzip with 'adm-zip' or Node.js built-in 'zlib'/'stream/promises',
  //      extract 3dmodel.model XML, parse <vertices> and <triangles> elements,
  //      same signed tetrahedron volume calculation
  //    NOTE: Use pure Node.js/TypeScript — no native binaries, no WASM in this plan.
  //      The signed tetrahedron method is ~30 lines of math, no library needed.
  // 3. Update ModelFile: { volumeMm3, surfaceAreaMm2, boundingBoxX, boundingBoxY, boundingBoxZ,
  //                         triangleCount, status: 'ANALYSIS_COMPLETE' }
  // 4. Enqueue: quoteComputeQueue.add('compute', { uploadId, modelFileId, mode: 'instant' },
  //      { delay: 0, attempts: 2 })
  // 5. On any parse error: set ModelFile.status = 'ANALYSIS_FAILED',
  //      enqueue quoteComputeQueue with mode: 'manual_review'
}, { connection: redis, concurrency: 2 })
```

Volume formula (signed tetrahedron method — no library needed):
```typescript
function computeVolumeMm3(triangles: Array<[v0: [number,number,number], v1: [number,number,number], v2: [number,number,number]]>): number {
  let vol = 0
  for (const [v0, v1, v2] of triangles) {
    // signed volume of tetrahedron from origin to triangle
    vol += (v0[0] * (v1[1] * v2[2] - v1[2] * v2[1]) +
            v0[1] * (v1[2] * v2[0] - v1[0] * v2[2]) +
            v0[2] * (v1[0] * v2[1] - v1[1] * v2[0])) / 6.0
  }
  return Math.abs(vol)
}
```

**B. Quote compute worker (apps/worker/src/processors/quote-compute.ts)**

BullMQ Worker consuming `'quote-compute'` queue, concurrency 5:

```typescript
new Worker('quote-compute', async (job) => {
  const { uploadId, modelFileId, mode } = job.data

  if (mode === 'manual_review') {
    // Create Quote with status 'MANUAL_REVIEW', no line items computed
    await prisma.quote.create({ data: {
      tenantId, uploadId, modelFileId, status: 'MANUAL_REVIEW',
      estimateType: 'MANUAL', totalCents: 0,
      lineItems: [], expiresAt: addDays(new Date(), 7)
    }})
    // Notify customer via notifyQueue
    return
  }

  // Instant quote path
  const modelFile = await prisma.modelFile.findUniqueOrThrow({ where: { id: modelFileId } })
  const rules = await prisma.pricingRuleSet.findFirst({
    where: { tenantId, isActive: true }, orderBy: { updatedAt: 'desc' }
  })

  // Pricing formula (configurable via PricingRuleSet):
  // baseCostCents = Math.ceil(volumeMm3 / 1000 * rules.pricePerCm3Cents)
  // setupFeeCents = rules.setupFeeCents (flat per quote)
  // materialCostCents = baseCostCents * rules.materialMultiplier
  // supportCostCents = (surfaceAreaMm2 > rules.supportThresholdMm2)
  //   ? Math.ceil(surfaceAreaMm2 / 100 * rules.supportPer100mm2Cents)
  //   : 0
  // totalCents = setupFeeCents + materialCostCents + supportCostCents
  // Apply manualReviewThreshold: if totalCents > rules.manualReviewAboveCents → manual_review
  // Apply geometry flag: if boundingBox dimension > rules.maxDimensionMm → manual_review

  const lineItems = [
    { label: 'Material cost', cents: materialCostCents, note: `${(volumeMm3/1000).toFixed(1)} cm³` },
    { label: 'Setup fee', cents: setupFeeCents },
    ...(supportCostCents > 0 ? [{ label: 'Support structures', cents: supportCostCents }] : []),
  ]

  await prisma.quote.create({ data: {
    tenantId, uploadId, modelFileId,
    status: isManualReview ? 'MANUAL_REVIEW' : 'INSTANT_ESTIMATE',
    estimateType: isManualReview ? 'MANUAL' : 'INSTANT',
    lineItems: JSON.stringify(lineItems),
    totalCents,
    volumeMm3: modelFile.volumeMm3,
    expiresAt: addDays(new Date(), 7),
  }})

  // Update Upload.status to 'QUOTED'
  await prisma.upload.update({ where: { id: uploadId }, data: { status: 'QUOTED' } })

  // Notify customer: notifyQueue.add('quote-ready', { quoteId, userId, estimateType })
}, { connection: redis, concurrency: 5 })
```
