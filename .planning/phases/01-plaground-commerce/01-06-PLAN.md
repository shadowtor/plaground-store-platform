**B. Orders API routes (apps/api/src/routes/v1/orders/index.ts)**

`POST /api/v1/orders` — optional auth (guest or customer). Body:
```typescript
{
  items: Array<{ productVariantId: string, quantity: number }>,
  customerEmail: string,  // required for guests; pre-filled for authenticated users
  shippingAddress: { line1, line2, city, state, postcode, country },
  paymentProvider: 'stripe' | 'paypal',
}
```

Logic:
1. Validate all `productVariantId` values exist and are `inStock`.
2. Compute total: sum of `productVariant.priceCents * quantity` for each item.
3. Create `Order` record: `{ tenantId, userId: session?.userId ?? null, status: 'CREATED',
   customerEmail, shippingAddress: JSON.stringify(shippingAddress), totalCents }`.
4. Create `OrderItem` records for each line item.
5. Create `OrderEvent` record: `{ orderId, type: 'PLACED', note: 'Order placed by customer' }`.
6. If `paymentProvider === 'stripe'`:
   - Call `createStripePaymentIntent({ orderId, amountCents: total, idempotencyKey: orderId })`.
   - Create `Payment` record: `{ orderId, provider: 'STRIPE', providerRef: paymentIntentId,
     status: 'PENDING', amountCents: total }`.
   - Return `{ orderId, paymentProvider: 'stripe', clientSecret }`.
7. If `paymentProvider === 'paypal'`:
   - Call `createPayPalOrder({ orderId, amountCents: total })`.
   - Create `Payment` record: `{ orderId, provider: 'PAYPAL', providerRef: paypalOrderId,
     status: 'PENDING', amountCents: total }`.
   - Return `{ orderId, paymentProvider: 'paypal', approveUrl }`.

`GET /api/v1/orders` — authenticated, `orders.read`. Cursor pagination (`?cursor&limit=20`).
Returns array of `{ id, status, totalCents, customerEmail, createdAt, itemCount }`.

`GET /api/v1/orders/:id` — authenticated, `orders.read`. Returns full order with `OrderItem[]`,
`OrderEvent[]` (status timeline), `Payment` status, and `Invoice.pdfStorageKey` (for receipt
download link generation).

`GET /api/v1/orders/:id/receipt` — authenticated, `orders.read`. Generates a short-lived (1hr)
presigned GET URL for the invoice PDF from S3. Returns `{ downloadUrl: string, expiresAt: string }`.
If `invoice.pdfStorageKey` is null (PDF not yet generated), return 202 with
`{ status: 'PENDING', message: 'Receipt is being prepared. Try again in 30 seconds.' }`.

`POST /api/v1/orders/:id/reorder` — authenticated, `orders.write`. Finds previous order, creates
a new order with the same items and `status: 'CREATED'`, returns new `orderId` without payment.
Customer initiates checkout from the cart with the pre-filled items.

**C. Stripe webhook handler (apps/api/src/routes/v1/payments/webhooks-stripe.ts)**

Route: `POST /api/v1/payments/webhooks/stripe`
Config: `{ rawBody: true }` — REQUIRED for signature verification (per CLAUDE.md Pitfall 4).
Register `@fastify/rawbody` plugin before this route in the plugin registration order.

```typescript
fastify.post('/payments/webhooks/stripe', { config: { rawBody: true } }, async (req, reply) => {
  const sig = req.headers['stripe-signature'] as string
  let event: Stripe.Event
  try {
    event = stripe.webhooks.constructEvent(req.rawBody!, sig, STRIPE_WEBHOOK_SECRET)
  } catch {
    return reply.status(400).send({ error: 'Invalid signature' })
  }

  // Replay protection: check EventLog table for idempotency
  const existing = await prisma.processedWebhookEvent.findUnique({
    where: { providerEventId: event.id }
  })
  if (existing) return reply.status(200).send({ status: 'already_processed' })

  // Record the event immediately to prevent concurrent processing
  await prisma.processedWebhookEvent.create({
    data: { providerEventId: event.id, provider: 'STRIPE', processedAt: new Date() }
  })

  switch (event.type) {
    case 'payment_intent.succeeded': {
      const pi = event.data.object as Stripe.PaymentIntent
      await prisma.payment.updateMany({
        where: { providerRef: pi.id },
        data: { status: 'CAPTURED', capturedAt: new Date() }
      })
      // Fetch order, update status to 'PAID', create OrderEvent, enqueue PDF receipt job
      const payment = await prisma.payment.findFirst({ where: { providerRef: pi.id } })
      if (payment) {
        await prisma.order.update({ where: { id: payment.orderId }, data: { status: 'PAID' } })
        await prisma.orderEvent.create({ data: {
          orderId: payment.orderId, type: 'PAID', note: 'Payment captured via Stripe'
        }})
        await notifyQueue.add('order-status-change', { orderId: payment.orderId, newStatus: 'PAID' })
        await pdfReceiptQueue.add('generate', { orderId: payment.orderId }, { attempts: 3 })
      }
      break
    }
    case 'payment_intent.payment_failed': {
      const pi = event.data.object as Stripe.PaymentIntent
      await prisma.payment.updateMany({
        where: { providerRef: pi.id }, data: { status: 'FAILED' }
      })
      break
    }
    // Add payment_intent.amount_capturable_updated for auth window tracking
  }

  return reply.status(200).send({ received: true })
})
```

**A. PDF receipt worker (apps/worker/src/processors/pdf-receipt.ts)**

BullMQ Worker consuming `'pdf-receipt'` queue, concurrency 3.

Use `@react-pdf/renderer` to generate a styled PDF receipt. Install it in `apps/worker/package.json`.

```typescript
new Worker('pdf-receipt', async (job) => {
  const { orderId } = job.data

  // Fetch order with items and payment
  const order = await prisma.order.findUniqueOrThrow({
    where: { id: orderId },
    include: { orderItems: { include: { productVariant: { include: { product: true } } } }, payment: true }
  })

  // Generate PDF using @react-pdf/renderer
  // Document structure:
  // - Header: PLAground logo text + "Receipt" heading
  // - Order details: Order ID, Date, Customer email
  // - Line items table: Product name | Qty | Unit price | Line total
  // - Separator
  // - Total row
  // - Payment method and status
  // - Footer: "Thank you for your order." + contact email
  const pdfBuffer = await renderToBuffer(<ReceiptDocument order={order} />)

  // Upload to S3
  const storageKey = `receipts/${order.tenantId}/${orderId}.pdf`
  await s3Client.send(new PutObjectCommand({
    Bucket: S3_BUCKET_NAME, Key: storageKey,
    Body: pdfBuffer, ContentType: 'application/pdf',
    ContentDisposition: `attachment; filename="receipt-${orderId}.pdf"`,
  }))

  // Create or update Invoice record
  await prisma.invoice.upsert({
    where: { orderId },
    create: { tenantId: order.tenantId, orderId, pdfStorageKey: storageKey,
              generatedAt: new Date(), amountCents: order.totalCents },
    update: { pdfStorageKey: storageKey, generatedAt: new Date() }
  })
}, { connection: redis, concurrency: 3 })
```
